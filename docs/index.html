台積電的護城河在於 EDA/IP 生態系

SemiAnalysis 發布了一則長串推文（1/8 至 8/8），核心論點是：台積電真正的護城河並非製程效能（PPA）、EUV 或良率，而是包覆在晶圓廠周圍的 EDA/IP 生態系 [10]。他們指出台積電的開放創新平台（OIP）已把 Synopsys、Cadence、Arm、Rambus、Alphawave 等數十家 IP 供應商，轉化為一個預先驗證的 tape-out 網路，而且這道護城河是可量化的 [9]。從數據來看，台積電認證的矽智財（Silicon IP / SiP）庫存項目，從 2010 年約 3,000 項成長到 2025 年的 93,000 項，15 年間成長達 31 倍 [8]，附圖也逐年呈現此成長趨勢與年增率變化 [10]。

他們進一步分析，每多一個預先驗證的 SerDes、HBM、PCIe、UCIe、記憶體與 chiplet 介面模組，就降低了在台積電流片的風險，同時提高把旗艦 ASIC 轉移到三星或 Intel 晶圓廠的成本 [7]。客戶不會單因競爭對手宣稱更好的 PPA 就更換晶圓廠，只有當整個設計風險堆疊都能一起移動時才會轉換——這才是晶圓廠的護城河 [6]。這個關係是雙向的：晶圓廠用 EDA/IP 生態系加深平台鎖定，EDA 廠商則透過晶圓廠認證取得與設計者的第一線接觸機會 [5]。因此，三星與 Intel 追趕的不只是台積電的製程路線圖，更是這個 TSMC-EDA/IP 的回饋循環 [4]。

GB300 NVL72 的推論效能

SemiAnalysis 也點名 NVIDIA、SGLang 與 RadixArk 團隊的成果，指出在推論互動性（interactivity）上，最高可比 B300 快 3.7 倍 [2]。附圖以 DeepSeek V4 Pro 1.6T（FP4、8K/1K）為基準，比較每 GPU token 吞吐量與互動性的關係，顯示搭配 Dynamo SGLang、MTP 的 GB300 NVL72 在中低互動性區間吞吐量明顯領先 B300 [2]。
